Text Data Cleaning Library

TextCleaner is an open source, MIT licenced library for text data pre-processing.
Ready to make the text cleaning hassle free.

pip install textcleaner

Motto

Open source community is growing day by day. Developers are using and utilizing every possible peace of code available on the internet. The main force which drives this community is contribution made by developers and researchers. There are many ways to contribute to this community like writing a good documentation, a readable code, re-usable codes, reporting issues and many more.

Like mentioned above, textcleaner is a little contribution to this vast community. This utility library is currently under development. However, the available version is capable of executing all the basic and some advanced cleaning proceses. It is structured using object oriented programming concepts to make it highly re-usable and enables features like chain calling, data handling by object, storing metadata which are very useful to the developers dealing with text data.
